Comandos mas utilizados en GIT

  • git init:Inicializar un repositorio local.
  • git add:Agrega archivos al area de espera.
  • git status:Muestra el estado del repositorio.
  • git log:Muestra el historial de los commits.
  • git commit: Captura el estado del repositorio en un punto del tiempo.
  • git tag:Cambia el id por una etiqueta.
  • git checkout:Permite viajar por los commits.
  • git branch:Permite crear ramas.
